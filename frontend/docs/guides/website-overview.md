## ğŸ¨ Customization Guide

### Changing Brand Colors

Edit CSS custom properties in `src/styles/globals.css`:

```css
:root {
  --primary-color: #ff6b35; /* Mango Orange */
  --secondary-color: #228b22; /* Natural Green */
  --accent-color: #8b4513; /* Jaggery Brown */
  /* ... */
}
```

### Updating Product Content

- **Product Information**: Edit product arrays in `Hero.tsx`
- **Product Images**: Replace images in `/public/images/` directory
- **Product Descriptions**: Update product details and features
- **Pricing**: Add pricing information to structured data

### Customizing SEO

- **Meta Tags**: Edit SEO configurations in `src/utils/seo.ts`
- **Structured Data**: Update Schema.org markup in components
- **Keywords**: Modify target keywords in SEO utility functions
- **Social Media**: Update social media links in `src/constants/socialMedia.ts`

### Adding New Sections

1. Create new component in `src/components/common/NewSection/`
2. Add TypeScript interfaces in `src/types/`
3. Create CSS styles with proper naming
4. Add SEO optimization with `useSEO` hook
5. Export from `index.ts` and import in `HomePage.tsx`
