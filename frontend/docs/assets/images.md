# Social Media Assets

This directory contains optimized images for social media sharing and SEO purposes.

## Image Requirements

### Open Graph Images

- **Size**: 1200x630 pixels (recommended)
- **Format**: PNG or JPG
- **File size**: Under 8MB
- **Aspect ratio**: 1.91:1

### Twitter Card Images

- **Size**: 1200x675 pixels (recommended)
- **Format**: PNG, JPG, or WebP
- **File size**: Under 5MB
- **Aspect ratio**: 16:9

### Favicons

- **Sizes**: 16x16, 32x32, 48x48, 192x192, 512x512
- **Format**: PNG or ICO
- **Background**: Transparent or brand color

All images should include:

- Descriptive alt text
- Proper file naming (kebab-case)
- Optimized file sizes for web
- Responsive sizing options
